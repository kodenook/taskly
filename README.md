
# Taskly

Organize and manage your tasks quickly and easily.


## Features

### Task
- `create`
- `list`
- `remove`

## Tech Stack

**Client:** Nextjs, TailwindCSS, shadcn


## Run Locally

Clone the project

```bash
  git clone https://github.com/kodenook/taskly.git
```

Go to the project directory

```bash
  cd taskly
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
