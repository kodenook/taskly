## Features

- Add task
- Check task as completed
- Delete task


## Run Locally

Clone the project

```bash
  git clone -b main https://github.com/kodenook/taskly.git
```

Go to the project directory

```bash
  cd taskly
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve
```
